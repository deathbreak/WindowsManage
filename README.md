# WindowsManage
写的一个运行exe的url管理器
#### 1.建议用tabby等ssh工具打开
#### 2.数据库使用的是python自带的sqlite
#### 3.windowsmanage.py是主程序
###### 需要使用pyinstaller打包成exe再使用(打包命令-->pyinstaller -F xxx.py -i xxx.ico)
